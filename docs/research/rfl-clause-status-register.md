# RFL Clause Status Register

This page hardens item 3 by separating RFL status, Championship relevance, record visibility, and validation gaps. It should be read with [RFL Clause Evidence Map](rfl-clause-evidence-map.md).

The register is based on the [RFL Medical Standards 2026](https://www.rugby-league.com/uploads/docs/Medical%20Standards%202026.pdf). It is a research register, not a substitute for the RFL source document.

## Status definitions

| Status | Meaning for this research |
| --- | --- |
| Mandatory | The RFL standards expressly identify the area as mandatory or mandatory in substance. |
| Competition-specific mandatory | Mandatory for specified competitions, teams, roles, or circumstances. |
| Best practice | The RFL standards present the area as best practice or recommended. |
| Information only | The RFL standards present the area as information or guidance rather than a direct action requirement. |
| Mixed | The section contains mandatory and non-mandatory elements that need separating before final evidence design. |
| Needs validation | The source status is not yet sufficiently classified for a club-specific pack. |

## Championship-specific research notes

- Cardiac screening is identified in the RFL standards as mandatory for specified competitions including Championship, but provider/control evidence still needs a CQC diagnostic/screening review.
- Mandatory medical equipment and drugs box requirements apply across club contexts; exact home, away, training, academy, reserves, and scholarship responsibilities need extracting into a club-specific checklist.
- Championship match-day cover requirements appear in competition tables; these should be converted into a short Championship cover register before any implementation pack is finalised.
- Some controls are RFL best practice but may still be locally necessary because they reduce CQC-style governance risk.
